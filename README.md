# Vrinda-Store-Sales-Analysis-Excel-Dashboard

## Project Overview
This project analyzes **Vrinda Store sales data** using Microsoft Excel to identify sales trends and customer purchasing behavior. The analysis is performed using data cleaning, pivot tables, and an interactive Excel dashboard.

## Tools Used
* Microsoft Excel
* Pivot Tables
* Data Cleaning
* Data Visualization / Dashboard

## Analysis Performed
* Monthly **Sales vs Orders** analysis
* **Men vs Women** purchasing comparison
* **Order Status** analysis (Delivered, Cancelled, Refunded)
* **State-wise sales** performance
* **Age and Gender** distribution
* **Sales Channel** performance (Amazon, Flipkart, etc.)

## Key Insights
* Female customers contribute more to total sales.
* Maharashtra and Karnataka generate the highest revenue.
* Most orders are successfully delivered.
* Online platforms like Amazon and Flipkart drive major sales.
